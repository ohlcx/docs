# stripe-credits-billing

**Composer:** `ohlcx/stripe-credits-billing` · **Repository:** [github.com/ohlcx/stripe-credits-billing](https://github.com/ohlcx/stripe-credits-billing) (private)

## Role

Monetization via Stripe Checkout: credit packages, consumption ledger, webhooks, admin billing adjustments.

## Host setup

```bash
php artisan stripe-credits-billing:seed
```

Configure Stripe keys in host `.env` (not documented here).
