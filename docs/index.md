# OHLCX Developer Hub

Welcome to the **public** documentation for OHLCX — a Laravel + React trading platform with Schwab integration, strategies, realtime features, **in-app AI**, and an **MCP server** for IDE automation.

!!! note "Docs-only repository"
    Application and Composer package **source is private**. This site explains architecture, APIs, setup (for authorized contributors), AI/MCP, and the package ecosystem.

## Editions

| Edition | Focus |
|---------|--------|
| **Pro** | Full local domain packages (strategies, sectors, pricefeed, news, analysis) |
| **Light** | Core trading UI + remote OHLCX API for sectors and domain data |

Open documentation lives here. **Source access** requires authorization from OHLCX maintainers and a GitHub token for Composer. See [Contributor access](getting-started/contributor-access.md).
